# Mohamed Hekal Portfolio

Professional portfolio for Mohamed Hekal, Full Stack AI Engineer.

## Includes

- CV and profile photo assets
- GitHub links for public source projects
- Screenshot-only case studies for private-source projects
- Project cards with validation/test evidence
- Responsive desktop and mobile layout

## Development

```bash
npm install
npm run dev
```

Open `http://127.0.0.1:4173`.

## Build

```bash
npm run build
npm run test:portfolio
```

## Screenshot Capture

```bash
npm run capture
```

The capture script starts local project servers where possible and saves screenshots to `public/screenshots/`.
